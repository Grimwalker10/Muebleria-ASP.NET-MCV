# ASP.NET MVC

<hr>

## Español:

## Proyecto ASP.NET MVC

Proyecto realizado con ASP.NET, bootstrap, JQuery, Ajax, con base de datos **Oracle 19c XE**

### Problema: 

La muebleria Los Alpes quiere tener un control de los productos y ventas que hace a nivel nacional e internacional de todas sus tiendas, 
se le encomienda hacer una paguina web en la cual estan registrados los empleados, informacion de los productos, proveedores y el reporte de ventas.

### Caracteristicas:

El proyecto esta divido en varias partes:
```
CapaDatos
CapaEntidad
CapaNegocio
CapaPrensentacionAdmin (FrontEnd)
CapaPresentacionTienda (Admin)
```

<hr>

## English:

## ASP.NET MVC Project

Project made with ASP.NET, bootstrap, JQuery, Ajax, with **Oracle 19c XE** database

### Problem: 

Los Alpes furniture store wants to have control of the products and sales it makes nationally and internationally in all its stores, 
You are entrusted with making a web page in which employees, product information, suppliers and sales reports are registered.

### Characteristics:

The project is divided into several parts:
```
DataLayer
EntityLayer
LayerBusiness
PresentationLayerAdmin (FrontEnd)
LayerPresentationStore (Admin)
```
